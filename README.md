# DotWebStack Web Components

A set of embeddable Linked Data web components, built on React.

## Prerequisites

The following tools are required:

* [Node.js v8 (LTS)](https://nodejs.org/en/)

## Development

Install dependencies:

```
npm install
```

Start development server:

```
npm run start
```

## Release

Build:

```
npm run build
```
